## Purpose
팀원 빌딩을 위한 플랫폼 및 학습 컨텐츠를 제공하는 웹 페이지 구축


## Description
프로젝트 팀원 빌딩에 있어서 좀 더 객관적인 지표로 팀원을 구성하기 위해서 프로젝트를 시작하게 되었다. 

추가적으로, 코딩 학습에 있어서 도움이 될만한 학습 컨텐츠를 제공할 예정이다.

## Stacks

Backend : Django, Ajax, jQuery

Frontend : HTML, CSS, Javascript, Bootstrap
